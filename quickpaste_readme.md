# 이 프로젝트가 해결하려는 문제
모바일개발자로 일하다보니 테스트폰과 컴퓨터 간 문자열을 주고 받을 일이 많음
디바이스의 토큰값을 컴퓨터로 전달해서 컴퓨터에서 postman등 툴로 검증을 해본다던지 컴퓨터에서 특정 링크를 디바이스에서 실행해 본다던지

그때마다 내 폰, 내 맥이면 에어드랍 등으로 간단히 해결가능함
하지만 테스트폰이나 다른 사람의 폰이나 컴퓨터의 경우 매개체가 필요함
보통 슬랙이나 이메일이 있는데 이 경우 로그인을 해야한다거나 
앱을 설치해야함

그래서 이 프로젝트는 그 불편함을 없앨 것임
